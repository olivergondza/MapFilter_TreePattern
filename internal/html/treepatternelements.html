TreePattern Elements

Pattern tree structure consists of several types of nodes and tree leaves.

Branching

There are four so-called Branching nodes (All, One, Opt and Some) that specifies data attribute relations.

All node

The All node needs all its followers satisfied to get satisfied itself.

Opt node

The Opt node hasn't got any conditions. Occurrence of its followers is completely optional.

One node

The One node needs exactly one satisfied follower.

Some node

The Some node needs at least one satisfied follower.

TreePattern Common Attributes

Following attributes are shared with all TreePattern elements.

Attribute flag

A flag attribute specifies a flag to set for the node if the node get satisfied.

Attribute assert

An assert attribute specifies an assertion to set for the node if the node won't get satisfied even though its parent was satisfied.

Attribute attachPattern

A attachPattern attribute specifies a pattern to attach into the node to become its regular content.

TreePattern Pattern attributes

Attributes specific for pattern node.

Attribute name

A name attribute specifies a name of certain pattern. The name is used when the pattern is attached to the node.
