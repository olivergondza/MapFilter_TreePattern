Parse .ini file example

This example is copied from the PHP manual (http://cz.php.net/manual/en/function.parse-ini-file.php).

There is an .ini file that is deserialized by standard function and then parsed by MapFilter.

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE pattern SYSTEM "TreePattern053.dtd">
<pattern>
  <all>
    <key name="first_section">

      <all assert="no_any_of_that">
        <key name="one" assert="no_one" />
        <key name="five" assert="no_five" />
        <key name="animal" assert="no_animal" />
      </all>

    </key>
    <key name="second_section">

      <all>
        <key name="path" />
        <key name="URL" />
      </all>

    </key>
    <key name="third_section">

      <key name="phpversion">
        <iterator />
      </key>

    </key>
  </all>
</pattern>

Source of original initialization file. 

; This is a sample configuration file
; Comments start with ';', as in php.ini

[first_section]
one = 1
five = 5
animal = Dodo bird

[second_section]
path = "/usr/local/bin"
URL = "http://www.example.com/~username"

[third_section]
phpversion[] = "5.0"
phpversion[] = "5.1"
phpversion[] = "5.2"
phpversion[] = "5.3"

Content of this file is considered valid according to the filter.
