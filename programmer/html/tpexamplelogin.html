Login example

Example of a fictive login service.

There are mandatory attributes name and pass that may be followed by optional attribute use-https. Optionally a remember attribute can be used but must be accompanied with either user or server attribute.

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE pattern SYSTEM "TreePattern053.dtd">
<pattern>
  <all flag="login">

    <key name="name" assert="no_name" />
    <key name="pass" assert="no_password" />

    <opt>

      <key name="use-https">
        <value pattern="yes" flag="use_https" />
      </key>
      
      <all flag="remember">

        <key name="remember" />
        <one assert="no_remember_method">

          <key name="user" />
          <key name="server" />

        </one>

      </all>

    </opt>

  </all>
</pattern>

There is a list of input arrays on the first row of each set and the result arrays on the second. Flags can be found on the third row and the assertions on the fourth.

        // Valid set; 'login' flag will be set
        Array (
            Array ( 'name' => 'me', 'pass' => 'myPass' ),
            Array ( 'name' => 'me', 'pass' => 'myPass' ),
            Array ( 'login' ),
            Array ()
        ),
        // Valid set; 'login' and 'use_https' flags will be set
        Array (
            Array ( 'name' => 'me', 'pass' => 'myPass', 'use-https' => 'yes' ),
            Array ( 'name' => 'me', 'pass' => 'myPass', 'use-https' => 'yes' ),
            Array ( 'login', 'use_https' ),
            Array ()
        ),
        // Valid set; 'login' and 'remember' flags will be set
        Array (
            Array ( 'name' => 'me', 'pass' => 'myPass', 'remember' => 'yes', 'server' => NULL ),
            Array ( 'name' => 'me', 'pass' => 'myPass', 'remember' => 'yes', 'server' => NULL ),
            Array ( 'login', 'remember' ),
            Array ()
        ),
        // 'use-https' has got an invalid value
        Array (
            Array ( 'name' => 'me', 'pass' => 'myPass', 'use-https' => 'no', 'remember' => 'yes', 'server' => NULL ),
            Array ( 'name' => 'me', 'pass' => 'myPass', 'remember' => 'yes', 'server' => NULL ),
            Array ( 'login', 'remember' ),
            Array ()
        ),
        // 'use-https' has got an invalid value and one of 'user' and 'server' attributes is redundant
        Array (
            Array ( 'name' => 'me', 'pass' => 'myPass', 'use-https' => 'no', 'remember' => 'yes', 'server' => NULL, 'user' => NULL ),
            Array ( 'name' => 'me', 'pass' => 'myPass', 'remember' => 'yes', 'user' => NULL ),
            Array ( 'login', 'remember' ),
            Array ()
        ),
        // Since 'name' is a mandatory attributes the following set is invalid
        Array (
            Array (),
            null,
            Array (),
            Array ( 'no_name' )
        ),
        // Since 'password' is a mandatory attributes the following set is invalid
        Array (
            Array ( 'name' => 'me' ),
            null,
            Array (),
            Array ( 'no_password' )
        ),
        // None of 'user' and 'server' attributes was found;
        Array (
            Array ( 'name' => 'me', 'pass' => 'myPass', 'use-https' => 'yes', 'remember' => 'yes' ),
            Array ( 'name' => 'me', 'pass' => 'myPass', 'use-https' => 'yes' ),
            Array ( 'login', 'use_https' ),
            Array ( 'no_remember_method' )
        )
