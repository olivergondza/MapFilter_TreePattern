Direction example

In the following example there is a structure describing movement in certain direction ("north", "south", "east" and "west") and the distance defining a vector together.

Movement is described by one or two non-opposite directions. The distance can be determined by a "unit" and the amount of those units.

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE patterns SYSTEM "TreePattern053.dtd">
<!-- [TreePattern_Direction] -->
<patterns>
  <pattern>
    <some>
      <one>
        <key name="north" attachPattern="distance" />
        <key name="south" attachPattern="distance" />
      </one>
      <one>
        <key name="east" attachPattern="distance" />
        <key name="west" attachPattern="distance" />
      </one>
    </some>
  </pattern>
  <pattern name="distance">
    <all>
      <key name="count">
        <value pattern="/^\d+$/" />
      </key>
      <key name="unit">
        <value pattern="/^(?:meter|yard)$/" />
      </key>
    </all>
  </pattern>
</patterns>
<!-- [TreePattern_Direction] -->

There is a list of input arrays on the first row of each set and the result arrays on the second.
