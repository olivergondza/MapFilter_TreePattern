Filter example

Structure of filter utility.

-h and -v are completely optional options.

-c reverses column order -l reverses line order -a reverse character order in whole file.

While -c and -l can occur simultaneously -a is completely incompatible with other types of modifications.

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE pattern SYSTEM "TreePattern053.dtd">
<pattern>
  <one>

    <key name="-h" />
    <key name="-v" />

    <some>
      <key name="-c" />
      <key name="-l" />
    </some>

    <key name="-a" />

  </one>
</pattern>

There is a list of input arrays on the first row of each set and the result arrays on the second.
