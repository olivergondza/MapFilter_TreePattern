A new Pattern can be created from xml file (MapFilter_TreePattern_Xml::fromFile ( $url );) or xml in String (MapFilter_TreePattern_Xml::load ( $xmlString );).

TreePattern Result

TreePattern currently provides three different types of results. Result object can be obtained by fetchResult() from MapFilter instance. Individual results can be obtained by getMyResultType().

$filter->fetchResult()->getAsserts();

Results

Results can be obtained by getResults(). Results data structure contains a derivation of user query created according to given pattern.

Flags

Flags can be obtained by getFlags(). Flags represents predefined aspects where user data match expected data pattern.

Assertions

Assertions can be obtained by getAsserts(). Assertions represents predefined aspects where user data differs from expected data patern and related information describing the failures.
